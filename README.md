# Fault Diagnosis of Ball Bearings using Machine Learning Algorithms

## Abstract
Predictive maintenance is a crucial strategy for enhancing the operational reliability
and availability of machines across various industrial domains. It enables the mitiga
tion of unforeseen breakdowns and malfunctions in various machine components and
subcomponents, such as rotating elements like bearings. Since bearing failure modes
are invariably associated with elevated levels of machine vibrations, this parameter
is the predominant and most prevalent indicator for assessing bearing conditions and
prognosticating failures in the bearings.

---

## Objectives
Objectives of the project are to:
1. To identify if the bearing is faulty or healthy.
2. To compare Machine Learning techniques.
3. To produce a pure vibrational dataset from the bearings.
4. To evaluate statistical features from the vibration dataset.
5. To compare the efficacy of time-domain and frequency-domain data.
